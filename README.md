## 📌 Overview
This repository contains the projects, architectural designs, and optimized prompts developed during my **Prompt Engineering Internship at SkillCraft Technology**. The focus of this internship was on engineering robust interactions with Large Language Models (LLMs), developing Retrieval-Augmented Generation (RAG) pipelines, and evaluating generative outputs for production-grade AI applications.

## 🎯 Project Objectives
- **Design and Optimize Prompts:** Construct zero-shot, few-shot, and Chain-of-Thought (CoT) prompts to maximize LLM accuracy and reduce hallucinations.
- **Develop RAG Systems:** Integrate vector databases to ground LLM responses in factual, domain-specific data.
- **Implement Agentic Workflows:** Build autonomous LLM agents capable of utilizing external tools and APIs.
- **Cost & Latency Optimization:** Reduce token consumption and response latency without compromising output quality.
- **Evaluation & Benchmarking:** Establish robust evaluation frameworks using LLM-as-a-judge and standard NLP metrics.

## 🛠️ Technologies & Frameworks Used
- **Core:** Python, Jupyter Notebooks
- **AI/LLMs:** OpenAI API (GPT-4o, GPT-3.5-Turbo), Anthropic Claude, Hugging Face
- **Frameworks:** LangChain, LlamaIndex
- **Vector Databases:** ChromaDB, Pinecone
- **Tools:** Git, GitHub Actions, Docker, Streamlit (for rapid UI prototyping)

## 🧠 Core Skills & Competencies Acquired
- **Advanced Prompting:** Chain-of-Thought, Tree-of-Thoughts, and ReAct methodologies.
- **Context Window Management:** Chunking strategies, summarization loops, and token-aware text processing.
- **System Architecture:** Designing scalable middleware for LLM applications.
- **AI Safety & Alignment:** Implementing guardrails to prevent prompt injection and biased outputs.
- **Performance Tuning:** Hyperparameter tuning (Temperature, Top-P, Frequency Penalty) for specific use cases.

## 🚀 Getting Started

### Prerequisites
- Python 3.10+
- OpenAI API Key (or other supported LLM provider keys)

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/skmdshariff143-ai/SkillCraft-Prompt-Engineering.git
   cd SkillCraft-Prompt-Engineering
   ```
2. **Create a virtual environment:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```
3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
4. **Environment Setup:**
   Copy the example environment file and add your API keys.
   ```bash
   cp .env.example .env
   ```
